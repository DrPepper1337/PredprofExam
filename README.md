# Работа с базой препаратов

## В проекте реализованы следующие функции:
- Создание таблицы формата: месяц, количество препаратов, созданных в этот месяц
- Поиск по файлу, используя введённое название препарата
- Создание файла с уникальными идентификаторами для каждого препарата
- Преобразование исходных данных в hash-таблицу, функции для поиска по ней и добавления нового элемента
## Установка и запуск проекта
### Чтобы установить проект нужно ввести в консоль команду:
git clone https://github.com/DrPepper1337/PredprofExam
### Чтобы запустить какой-либо файл, нужно ввести команду:
python3 <имя файла>
## Как использовать проект:
- Файл task_1 можно использовать для создания из исходных данных, таблицу формата:  месяц, количество препаратов, созданных в этот месяц
- Файл task_3 можно использовать для того, чтобы получать информацию о препарата с введённым названием
- Файл task_4 можно использовать для того, чтобы создать файл с уникальными идентификаторами для каждого препарата
- В файле task_5 реализовано создание hash-таблицы из вводных данных, а также функции для поиска по ней и добавление туда нового элемента (чтобы использовать функции, необходимо написать в конец файла название функции и ввести в её тело необходимые аргументы, после чего запустить файл)
